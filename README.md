Instalacja

1. Sklonuj repozytorium
2. Zastosuj migracje bazy danych
dotnet ef database update --context ApplicationDbContext
Uruchom aplikację

Funkcje

- Rejestracja i logowanie
- Tworzenie sesji treningowych
- Dodawanie wykonanych ćwiczeń (obciążenie, serie, powtórzenia)
- Gotowe typy ćwiczeń + możliwość dodawania własnych
- Statystyki z ostatnich 4 tygodni
- Historia treningów
